愉快地安装和使用 VEGAS
==============
### Step One——Cross 404
* 下载<u>***CROSS404.7z***</u>，把里面的文件全部解压出来备用
* 安装里面的<u>***Chrome***</u>插件 (自行百度)
* 打开<u>***Account.txt***</u>，打开插件，待出现登录框后输入账号密码登录，随便选择一条线路，使浏览器联上国际网络
* 打开<u>***Link.txt***</u>，打开内部的网址注册一个账号，在邮箱中激活这个账号，下载<u>***Windows***</u>客户端，用注册好的账号登录，选择一条免费线路，使 Windows 联上国际网络
##### Done(1/2)!
###  
###  
### Step Two——正事  
* **(全程保持联上国际网络!!!)**
* 回到 “VEGAS” 文件夹，根据自己的需要下载对应版本的VEGAS下载器和补丁( 这里我只提供<u>***VEGAS 15&16***</u>系列**PRO&Suite**版本的官方下载器 )
* 根据自己的需要安装VEGAS及附加组件
* 安装汉化补丁(虽然用15版本的下载器可以下载到简体中文试用版,但是补丁用了都说好)
##### Done(2/2)!
